VS Code —- 前端常用插件以及配置

2018-01-27

官网下载地址 (打开网址自动下载)

  • 通用

    • Auto Close Tag 自动补全标签
    • Auto Rename Tag 自动更改标签名
    • Bracket Pair Colorizer 括号颜色拓展 每对括号为同一颜色
    • Path Intellisense 路径自动补全
    • beautify 格式化代码工具
    • Dash 查文档必备,快捷键 ctrl + h
    • fileheader 顶部注释模板，可定义作者、时间等信息，并会自动更新最后修改时间
    • filesize在底部状态栏显示当前文件大小，点击后还可以看到详细创建、修改时间
    • Code Runner运行多种语言的代码片段或代码文件，安装完成后，右上角出现箭头，点击箭头即可运行文件
    • Image Preview 鼠标移到路径里显示图像预览
    • Live Server 端口占用，热更新

  • HTML

    • HTML Snippets 超级实用且初级的 H5代码片段以及提示
    • Class autocomplete for HTML智能提示HTML class = “ ” 属性
    • htmltagwrap 可以在选中HTML标签中外面套一层标签

  • CSS

    • HTML CSS Support 让 html 标签上写class 智能提示当前项目所支持的样式
      • 新版已经支持scss文件检索
    • css peek能够查看CSS ID和类的字符串作为HTML文件中相应的CSS定义
      • 使用方法：将鼠标放在class属性，右键“查看定义”

  • JS

    • Document thisjs 的注释模板
      • 新版的vscode已经原生支持,在function上输入/** tab
    • ESlint 自定制提示规则
      • 配置链接
    • JavaScript (ES6) code snippets es6代码片段
    • JavaScript Snippet Pack js代码片段
    • Quokka.js实时观看 javascript 的变量的变化

  • JQ

    • jQuery Code Snippets Snippets

  • BS

    • Bootstrap 3 Snippets Snippets
    • Bootstrap 4 & Font awesome snippets for Visual studio code Snippets

  • Node

    • Node.js Modules Intellisense 可以在导入语句中自动完成JavaScript / TypeScript模块。

  • Git

    • GitLens 丰富的git日志插件
    • Git History以图表的形式查看git日志

  • 主题更改

    • 左侧插件栏搜索Theme，下载喜欢的主题

  • 自动保存

    • 配置文件
